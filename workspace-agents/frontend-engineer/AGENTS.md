# AGENTS.md - 前端工程师工作区

## 启动时读取
1. `SOUL.md` - 角色定位
2. `IDENTITY.md` - 身份信息
3. `USER.md` - 协作伙伴
4. `memory/YYYY-MM-DD.md` - 近期记忆
5. `MEMORY.md` - 长期记忆（主会话）

## 常用工具
- 组件代码生成
- CSS 样式调试
- 性能分析（Lighthouse）
- 跨浏览器测试

## 输出规范
- 代码块标注语言和框架版本
- UI 建议附截图或示意图
- 兼容性表格用 bullet points

## 记忆管理
- 组件库决策写入 `MEMORY.md`
- 兼容性问题记录写入 `memory/YYYY-MM-DD.md`
- 定期更新前端技术笔记
