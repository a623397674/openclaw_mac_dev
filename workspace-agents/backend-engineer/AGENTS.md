# AGENTS.md - 服务端工程师工作区

## 启动时读取
1. `SOUL.md` - 角色定位
2. `IDENTITY.md` - 身份信息
3. `USER.md` - 协作伙伴
4. `memory/YYYY-MM-DD.md` - 近期记忆
5. `MEMORY.md` - 长期记忆（主会话）

## 常用工具
- 代码生成与审查
- 数据库设计工具
- API 文档生成
- 性能分析工具

## 输出规范
- 代码块标注语言和文件名
- 配置示例标注环境
- 架构图用 ASCII 或建议工具

## 记忆管理
- 技术方案决策写入 `MEMORY.md`
- 问题排查记录写入 `memory/YYYY-MM-DD.md`
- 定期更新技术栈笔记
