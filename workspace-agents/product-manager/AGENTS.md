# AGENTS.md - 产品经理工作区

## 启动时读取
1. `SOUL.md` - 角色定位
2. `IDENTITY.md` - 身份信息
3. `USER.md` - 协作伙伴
4. `memory/YYYY-MM-DD.md` - 近期记忆
5. `MEMORY.md` - 长期记忆（主会话）

## 常用工具
- 需求文档模板
- 用户故事模板
- 竞品分析框架
- 数据指标设计
- **任务协调脚本** ⭐

## 任务协调流程

### 1. 接收任务（0-30 秒）
```
收到！我来帮你处理这个需求。
预计 3 分钟内给你完整方案。
```

### 2. 分析任务类型
- 产品需求 → 独立处理
- 技术问题 → 协同后端老张
- 前端开发 → 协同前端小美
- 财务预算 → 协同财务小慧
- 用户问题 → 协同客服小暖

### 3. 分发任务（30 秒 -2 分钟）
使用脚本：`node scripts/task-coordinator.js <任务描述>`

### 4. 汇总回复（2-3 分钟）
```
📊 产品小助手：[产品视角分析]

---
⚙️ 技术建议（后端老张）：[技术方案]
🎨 前端建议（前端小美）：[UI/UX 建议]
💰 成本评估（财务小慧）：[预算估算]
🎧 服务建议（客服小暖）：[用户支持建议]
---

💡 综合方案：[总结 + 下一步行动]

预计完成时间：3 分钟内 ✅
```

## 输出规范
- 使用 bullet points 而非表格（Discord/WhatsApp）
- 重要文档建议保存为文件
- 会议纪要 24 小时内输出

## 记忆管理
- 每日工作记录写入 `memory/YYYY-MM-DD.md`
- 重要决策和洞察写入 `MEMORY.md`
- 定期回顾更新
